# The PageRank Algorithm

The PageRank algorithm is the most lucrative application of the theory of Markov chains in discrete time, which you started studying in the last block.  
PageRank is an algorithm used by Google Search to rank web pages in their search engine results.  
The patent for the algorithm was owned by Stanford University, which received 1.8 million shares in Google in exchange for the use of the patent.  
Stanford sold these shares in 2005 for $336 million.

Information on how the PageRank algorithm operates can be found on [Wikipedia](https://en.wikipedia.org/wiki/PageRank).  
The most important thing to understand is that page rankings are determined by finding the limiting stationary distribution for a Markov chain.
