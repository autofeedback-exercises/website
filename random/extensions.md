# Suggested extensions: Random variables

You can produce portfolio reports that do the types of analysis that I have presented in the example reports that I have shared with them.
If you are choosing to do this, please do not redo these analyses on a Bernoulli random variable.  It is relatively easy to do these analyses
for one of the other random variables that have been introduced in the programming exercises.  These random variables include:

* Binomial random variables
* Geometric random variables
* Uniform (discrete) random variables
* Uniform (continuous) random variables

It is not much more difficult to study other random variables that were not covered, for example:

* Negative binomial random variables
* Normal distribution
* Sums of multiple independent random variables of one of the types in the exercises
* Functions of the the types of random random variables in the exercises
* Functions of sums of random variables of one of the types in the exercises

Distributions that you also covered in the first year that are little bit harder to generate (but still not difficult) include:

* The exponential distribution
* The Erlang distribution
* The hypergeometric distribution

Try to be creative in what you submit if you can. You will understand more if you engage creatively with this material.
